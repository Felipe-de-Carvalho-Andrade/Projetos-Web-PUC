## 📌 Sobre o Projeto

Projeto desenvolvido para a disciplina **Desenvolvimento de Interfaces Web (DIW)** que consiste na criação de uma página web simples que apresenta um **perfil profissional**, contendo informações pessoais, habilidades, hobbies, contatos e projetos acadêmicos.

A página foi construída utilizando apenas **HTML5** para a estrutura e **CSS3** para a estilização.

O layout foi organizado em duas partes principais:

- **Sidebar (barra lateral)**: foto, contatos, habilidades e hobbies  
- **Área principal**: nome, descrição pessoal e projetos acadêmicos

## 🎯 Objetivo da Atividade

Praticar os conceitos básicos de desenvolvimento web, incluindo:

- Estrutura básica de uma página **HTML**
- Uso de **títulos, parágrafos e listas**
- Inserção de **imagens**
- Criação de **links**
- Aplicação de **estilos com CSS**
- Organização visual do conteúdo
