# Desenvolvimento Web Front-End — Projetos da Disciplina

Este repositório reúne os **sites e atividades práticas desenvolvidas durante a disciplina de Desenvolvimento Web Front-End** do **1º período de Sistemas de Informação da PUC Minas**.

Cada pasta presente neste repositório representa **uma atividade prática semanal**, onde são aplicados os conceitos estudados em aula sobre desenvolvimento de páginas web.

---

## Objetivo do Repositório

* Organizar os projetos desenvolvidos durante a disciplina
* Registrar a evolução no aprendizado de HTML e CSS
* Manter um histórico das atividades realizadas

---

## Estrutura do Repositório

Cada atividade está organizada em sua própria pasta.

Exemplo de estrutura:

```
/Projetos-Web-PUC
│
├── atividade-01
│   ├── index.html
│   ├── style.css
│   └── images
│
├── atividade-02
│   ├── index.html
│   ├── css
│   │   └── style.css
│   └── images
│
└── README.md
```
